CYBE465 Programming Assignment - README

About: 
This program is a profile management system that allows users to manage friends, lists, and 
pictures. It supports commands for adding friends, creating lists, posting pictures, and 
managing permissions. The program also includes functionality for reading and writing comments 
on pictures, as well as auditing actions.

How to use: 
1. Open a terminal with Python installed.
2. Run the program using the command: `python access.py <filename>`.
   - Replace `<filename>` with the name of a file containing commands to execute.
   - If no filename is provided, you can manually enter commands interactively.
3. Follow the on-screen instructions to interact with the program.

Requirements:
- Python version: 3.10.6 
- No additional dependencies are required.

Outputs:
- `audit.txt`: Logs all actions performed during the program's execution.
- `pictures.txt`: Stores information about pictures, their owners, associated lists, and permissions.
- `lists.txt`: Stores information about lists and their members.
- `friends.txt`: Stores the list of friends added to the profile.
